GOOGLE STEP INTERNSHIP

A 12 week software engineering internship at Google over the summer.

My role at Google:

My internship was from June 2021 to September 2021. I developed a CLI Python tool that helps simplify making efficient resource allocation and management decisions. My tool contains 3 commands, can use current or historical data, can add or move resources to try saving space. This project included backend development using SQL, RPC calls, Python libraries, data processing, code testing and documentation.
